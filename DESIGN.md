# Dirección Visual — La Perrera Club

## Lectura de marca

### XOSA

El editable `xosa.psd` funciona como logo ilustrado, no como palabra tipográfica. La forma tiene masa pesada, contorno sucio, bordes granulados y un gesto de sticker pegado sobre foto. Por eso el sitio debe renderizar XOSA como imagen, no recrearlo con una fuente.

Uso:

- Blanco roto sobre negro/foto.
- Rojo únicamente para golpes o estados de alto énfasis.
- Sombra/desfase mínimo en rojo y amarillo para conservar el lenguaje de impresión/flyer.

### Tipografía Area

Los archivos Area se usan como sistema:

- Area Bold: navegación, botones, encabezados funcionales, paneles.
- Area Regular: cuerpo y formularios.
- Area Italic: frases editoriales y acentos como `presenta`.

No usar una sans genérica como voz principal. Tampoco fingir el logo de La Perrera con una serif cualquiera; si no hay logotipo aislado, se usa Area y arte real.

### La Perrera cover

El cover 2026 define la paleta fuerte:

- Rojo impresión: `#f41422`
- Negro profundo: `#050504`
- Papel viejo: `#eadcc8`
- Amarillo club sucio: `#d5c04f`
- Verde/azul nocturno: `#0c1714`

La textura no es neón limpio; es tinta, ruido, escaneo, flyer y noche.

### Booking / show

El flyer de booking aporta:

- Foto de show real como centro visual.
- XOSA arriba como marca principal.
- Rojo inferior como spray/impresión.
- Información funcional directa.

La UI debe parecer herramienta de operación musical, no landing page de startup.

## Reglas de interfaz

- Menos tarjetas decorativas; más bandas, cortes, reglas, pósters y layouts editoriales.
- Botones rectos de 4px, no pills suaves.
- Dock inferior como control rápido, no como adorno.
- La página pública debe abrir con contenido real aunque Supabase esté vacío.
- Estados vacíos deben sentirse intencionales y administrables.
- El video se usa como atmósfera, con poster y `preload="metadata"`.
